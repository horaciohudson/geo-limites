# Imoveis e Arquivos Tecnicos

## Onde esse fluxo aparece no menu

Na secao `Fluxo Operacional`, o sistema exibe pelo menos estas entradas:

- Imoveis
- Arquivos Tecnicos
- Normas Aplicadas
- Visualizador
- Gerar Memorial

## Etapa 1: Imoveis

A area `Imoveis` e o ponto de partida para organizar os dados principais do trabalho. Ela deve ser usada para cadastrar ou revisar o imovel que servira de base para as etapas seguintes.

## Etapa 2: Arquivos Tecnicos

Em `Arquivos Tecnicos`, o usuario seleciona e organiza os arquivos que serao usados no processo. O sistema trabalha com foco especial em arquivos tecnicos associados ao fluxo do memorial.

## Relacao entre as etapas

Essas duas telas andam juntas:

- o imovel fornece o contexto do trabalho
- os arquivos tecnicos fornecem a base operacional para visualizacao e geracao

## Boa pratica operacional

Antes de seguir para o visualizador, confirme:

1. Se o imovel correto foi escolhido.
2. Se os arquivos esperados estao selecionados.
3. Se nao ha duplicidade desnecessaria.
4. Se os nomes dos arquivos permitem identificar o conteudo.

## Sinal importante do sistema

O menu mostra contagem de arquivos selecionados em `Arquivos Tecnicos`, o que ajuda a validar rapidamente se ha material suficiente para continuar.
