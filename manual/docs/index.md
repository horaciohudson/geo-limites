# Manual do Usuario do GeoLimites

## Objetivo deste manual

Este manual explica como operar o GeoLimites no dia a dia, do primeiro acesso ate a geracao do memorial e a administracao basica do ambiente.

## Para quem este material foi feito

### Operador tecnico
Usa o sistema para cadastrar imoveis, enviar arquivos tecnicos, visualizar dados e gerar memoriais.

### Gestor ou responsavel pela operacao
Acompanha configuracoes, usuarios, regras de uso e padronizacao do fluxo interno.

### Administrador do sistema
Configura empresa, SMTP, usuarios e aspectos de acesso do tenant.

## Mapa rapido do sistema

O menu lateral atual do GeoLimites esta organizado em tres grupos:

- Fluxo Operacional
- Preparacao Tecnica
- Conta e Acesso

Dentro deles, as rotas principais do sistema sao:

- Imoveis
- Arquivos Tecnicos
- Normas Aplicadas
- Visualizador
- Gerar Memorial
- Normas Tecnicas
- Importacao de Exemplo
- Templates
- Conta
- Administracao
