📋 Como Testar o Sistema

🎯 Acesso Rápido: Usuários administradores podem acessar o sistema financeiro através do menu lateral ou navegando diretamente para /financial

1. Testando a Interface

  1. Faça login como administrador
  2. Acesse o menu lateral e clique em "💰 Sistema Financeiro"
  3. Navegue pelas abas:
    • Dashboard: Visão geral com estatísticas
    • Saldo: Detalhes do saldo atual
    • Comprar Créditos: Formulário de compra
    • Transações: Histórico de movimentações
    • Compras: Gerenciamento de compras
    • Estatísticas: Relatórios detalhados

2. Testando Funcionalidades

  1. Widget de Saldo: Observe o widget na navbar que mostra o saldo atual
  2. Validação de Créditos: Tente gerar um memorial sem créditos suficientes
  3. Notificações: Verifique alertas de saldo baixo
  4. Responsividade: Teste em diferentes tamanhos de tela

3. Testando Componentes Isolados

Para testar componentes individualmente, você pode usar o componente FinancialTest:

  1. Importe o componente: import FinancialTest from './components/financial/FinancialTest';
  2. Adicione uma rota temporária: <Route path="/test-financial" element={<FinancialTest />} />
  3. Acesse /test-financial para testar cada componente isoladamente

4. Configurações Disponíveis

  • Pacotes de Crédito: Editáveis em config/creditPackages.ts
  • Temas: Personalizáveis em styles/themes.css
  • Formatação: Utilitários em utils/formatters.ts
💡 Dica: O sistema está totalmente integrado e pronto para uso. Todos os componentes são responsivos e seguem as melhores práticas de acessibilidade.
